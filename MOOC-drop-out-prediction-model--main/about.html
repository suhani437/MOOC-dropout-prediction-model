About This Project

This project aims to predict student dropout risk in Massive Open Online Courses (MOOCs) using Machine Learning and user interaction data. Early prediction enables instructors and educational institutions to provide timely intervention and improve student success.

Project Objective

The main objective is to analyze student activity logs and classify learners into:

  • Active – safe and regularly interacting
  • At Risk – likely to drop out

Dataset Description

The dataset contains MOOC learner activity features such as:

  • Navigation, video views, problem attempts, discussion participation
  • Early, mid, and late activity events
  • Study days (active days)
  • Time between first and last log (activity duration)
  • Total interaction count derived from multiple activity logs

Machine Learning Model Used

  • Algorithm: Random Forest Classifier
  • Train-test split: 80/20
  • Main prediction features: total_activity, early_mid_late, active_days, activity_duration, video, discussion, problem
  • Random Forest was chosen for its reliability and ability to handle non-linear patterns

System Architecture

The workflow of the platform:

  • User enters feature values in the prediction form
  • Flask backend processes the data
  • Machine learning model predicts risk and returns probability
  • Dashboard visualizes overall statistics and risk distribution

Key Features

  • Real-time dropout risk prediction
  • Smart input validation with red-warning for invalid entries
  • Interactive bar chart showing entered feature values
  • Dashboard with average analytics and risk distribution
  • User-friendly UI built with Bootstrap and smooth gradient design

Limitations

  • Model accuracy depends on the quality of dataset
  • Only selected features from MOOC logs are used
  • No real-time live MOOC integration (yet)

Future Scope

  • Add deep learning models (LSTM, GRU) for time-series prediction
  • Integrate real-time activity tracking from a live MOOC platform
  • Generate personalized recommendations for at-risk learners
  • Add an admin dashboard with login system
  • Expand dataset for more accurate predictions
